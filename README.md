# elk
This project contains *Vagrantfile* for
- ElasticSearch
- Logstash
- Kibana

with
- Filebeat and
- TopBeat

running in the same VM.

### Prerequisites:
- VirtualBox
- Vagrant

### Installation
- "cd" to this directory and
- run "vagrant up"

### Usage
To access Kibana dashboards, go to
http://192.168.33.90
